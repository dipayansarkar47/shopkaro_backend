## Built with...

- 🚀️ React for the UI
- 🏅️ Redux-Toolkit
- 💪️ PassportJS & Jwt for Authentication
- 💬 react-alert & react-spinner
- 💎️ Tailwind CSS for modern look
- ⏲️ NodeJS + ExpressJS
- 🔥 Admin Panel
- 📸 Add/Update/Delete Product
- 🎉️ and much more.
